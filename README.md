# ToDo List:

This is another To-Do List web app ;-)

## Application Stack:

* Frontend: Angular 4 + Material Design.
* Backend: Slim PHP + MySQL DB. [Go to repository](https://github.com/maurobonfietti/rest-api-slim-php).


## Execute Commands:

``` bash
npm install
npm start
# or npm run local
# or ng serve
```
