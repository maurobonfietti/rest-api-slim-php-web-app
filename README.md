# ToDo List:

This is another To-Do List web app, for do amazing stuff ;-)

## Application Stack:

* Frontend: Angular 4 + Angular Material Design.
* Backend: Symfony PHP Framework + MySQL DB. [Go to repository](https://github.com/maurobonfietti/todo-list-back).


## Execute Commands:

``` bash
npm install
npm start
# or npm run local
# or ng serve
```


## Enjoy Online Demo:
